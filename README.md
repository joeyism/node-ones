# ones
A simple tool to create an array with only 1's

## Installation
To install, run

    $ npm install --save ones

## Usage
To use, require it first by

    var ones = require("ones");

Then input the array length that you want

    ones() // []
    ones(0) // []
    ones(1) // [1]
    ones(5) // [1, 1, 1, 1, 1]
